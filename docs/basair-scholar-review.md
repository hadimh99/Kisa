# Basa'ir al-Darajat — Scholar Review Sheet

**Purpose:** open questions from the AI translation/ingestion of *Basa'ir al-Darajat* (Part 1)
that need a scholar's ruling before the text is considered authoritative. Each item gives the
reference, the Arabic in question, what we currently have, and the specific question. Please
write your answer on the **"➡ Ruling:"** line (or annotate the PDF/Arabic directly).

**Two standing, whole-book items** (apply to everything, no per-item entry needed):
1. **Arabic proofing.** All Arabic was transcribed by *visually reading* the scanned page
   images (the PDF's Arabic text layer is corrupted), so every narration's Arabic needs a
   proofing pass for letters/dots/diacritics.
2. **English review.** The English is a fresh re-translation (not the hubeali English).
   Please confirm meaning/accuracy, especially the items below.

Citations use the book's own numbering: **P1 = Part 1**, **Ch = chapter**, **H = hadith**,
**Rare H = the "Rare from the chapter" (Nawadir) appendix**.
